# 引言

[`Zino`][zino]致力于打造[Rust][rust]语言中最好用的企业级应用开发框架。
我们借鉴Node的[`Egg.js`][eggjs]以及Java的[`Spring Boot`][spring-boot]框架，
奉行『约定优于配置』的原则，提供开箱即用的功能模块，极大提高开发效率；
并通过与Rust的[`axum`][axum]、[`actix-web`][actix-web]等框架集成，打通社区生态资源。
